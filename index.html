Changelog

Beta 1.2 (Latest)
  • Added Trees to world generation.
  • Added a "Leaves" block.
  • Fixed a bug where breaking adjacent blocks would cause previously broken blocks to reappear.
  • Added void-proofing: breaking blocks now fills in adjacent empty space.
  • Added instant teleportation when falling into the void.
  • Made all generated terrain blocks breakable.
Beta 1.1
  • Fixed world regenerating every time the game is paused.
  • Optimized chunk generation to only create the world's surface, improving performance.
  • Added dynamic terrain generation, a chunk system, and a fog toggle.
  • Added this changelog and a version selector.
  • Added an asynchronous loading screen.
Beta 1.0
  • Initial release with flat world generation.
  • Basic block placing/breaking and player movement.
  • PC and Mobile controls.
  • AI Structure builder, world saving/loading.
